# Data Sources

이 저장소에는 GitHub Pages에서 대시보드를 실행하는 데 필요한 최종 정적 산출물만 포함한다. 원자료, API 원본 응답, 중간 산출물은 포함하지 않는다.

| 데이터 | 용도 | 출처 | 기준 |
| --- | --- | --- | --- |
| 택지지구 경계 및 토지이용계획도 | 지역 경계, 토지이용 구성 | LH 공공택지정보시스템, VWorld | 분석 시점 최신 가용 자료 |
| 연속지적도 및 GIS 건물정보 | 필지/건축물 지도, 건축물 용도 분석 | VWorld | 분석 시점 최신 가용 자료 |
| 건축물대장 표제부 | 연면적, 용도, 용적률 등 건축물 지표 | 국토교통부 건축HUB | 분석 시점 최신 가용 자료 |
| 인구·사업체·종사자 통계 | 인구사회 지표, 접근 가능 인구·종사자 | 통계청 SGIS | 2023년 중심 |
| 격자 경계 및 격자 통계 | 지하철역 750m 접근 인구·종사자 산정 | 통계청 SGIS | 2025 격자 경계 및 제공 통계 |
| 수도권 지하철 네트워크 | 지하철 최단시간 접근성 | 수업 제공자료 `subway_network.zip` | 제공 그래프 기준 |
| 버스 정류장 자료 | 버스 접근성 및 정류장 도달권 | 서울 열린데이터광장, 경기데이터드림 | 분석 시점 최신 가용 자료 |
| 도로망/배경지도 | 지도 표시 및 도로망 보조 분석 | OpenStreetMap | 분석 시점 최신 가용 자료 |

## 제외한 데이터

- 원본 SHP, SHX, DBF, PRJ, CPG
- 원본 CSV 대용량 파일
- API 원본 응답 전체
- 임시 다운로드 파일
- 중간 처리 산출물
- 분석 중 생성한 로그, 캐시, 캡처 이미지

## 재현 참고

전처리 스크립트는 `scripts/`에 두는 것을 원칙으로 한다. 단, 원자료는 용량과 라이선스 문제로 저장소에 포함하지 않으므로, 재실행 시 원자료를 각 출처에서 별도로 확보해야 한다.
